# ESERCIZIO DI MARKDOWN
## ULTIMO LAVORO VERSIONATO SU GIT
### **Esercizio YAML**

### Consegna dell'esercizio
Realizzare in **YAML**[^1] una rubrica dove si legga:

- il **nome** dell'alunno/a, 
- il voto **pratico**, 
- quello **teorico**, 
- se insufficiente calcolando la **media** e scrivendo sì/no
- un **commento** sulla prova sostenuta.

In seguito traducilo in **XML** con annesso **XSD**[^2].

### Domande sull'esercizio
1. Qual è un elemento chiave della sintassi di YAML?
- [] Si usano tag come `<tag>`
- [x] L'indentazione è significativa
- [] Si usano sempre parentesi graffe {}

2. Come si scrivono gli elementi in XML?
- [x] Con tag di apertura e chiusura, es. `<nome>...</nome>`
- [] Solo con l'indentazione
- [] Separando i campi con virgole

3. Che cosa specifica uno schema XSD per un documento XML?
- [x] La struttura, i nomi degli elementi e i tipi di dato validi
- [] Le regole di formattazione visuale del documento
- [] Il sistema di controllo versioni del file

[^1]: Puoi trovare la sintassi di YAML nelle slide su Classroom

[^2]: L'XSD deve descrivere la sintassi del documento XML
